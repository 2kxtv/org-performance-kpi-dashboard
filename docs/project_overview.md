# Project Overview

This project demonstrates an end-to-end data analyst workflow for monitoring organizational performance, operational risk, bottlenecks, and data quality.

## Business Problem

Organizations require visibility into operational performance, delay patterns, disruption causes, and data reliability.

## Solution

A Power BI dashboard was developed using SQL-prepared operational data to provide executive KPI insights and operational analysis.

## Dashboard Pages

1. Executive Overview
2. Performance Drivers
3. Bottlenecks & Root Cause
4. Data Quality & Trust

## Tools Used

- SQL
- Power BI
- DAX
- GitHub
