# Dashboard Design Notes

The dashboard uses a clean executive design style with a light lavender background, rounded KPI cards, soft shadows, and consistent purple branding.

## Design Choices

- Executive KPI cards for quick decision-making
- Driver charts to explain performance patterns
- Bottleneck visuals for root cause analysis
- Data quality page to prove trust and completeness

## Theme

- Background: #F5F3FC
- Primary Purple: #5B4BB7
- Medium Purple: #8F82D8
- Light Lavender: #D9D5F3
- Text: #1F2340
