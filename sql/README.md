# SQL Scripts

This folder contains SQL scripts used to prepare the operational dataset for Power BI reporting.

## Script Order

1. `01_create_tables.sql`  
   Creates the required database tables.

2. `02_load_clean.sql`  
   Loads and prepares the operational dataset.

3. `03_kpi_queries.sql`  
   Creates KPI queries and reporting views.

4. `04_data_quality_checks.sql`  
   Performs completeness and trust checks.
